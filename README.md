ShiftixApp is a mobile application designed to help users manage and organize their work shifts. The app allows users to submit shift requests, request shift swaps, view notifications, and track upcoming or previous shifts in a structured way.
It provides an easy-to-use interface with separate screens for adding shifts, managing requests, and viewing shift-related updates.
